mitmproxy & mitmdump 使用介绍

简介

mitmproxy和mitmdump是一个用于监视和修改http请求的工具,具体用来干什么就靠个人发挥 了. mitmproxy是带交互界面的, 而mitmdump是纯命令行.

官网

官网地址: http://mitmproxy.org

Github: https://github.com/cortesi/mitmproxy

下载&安装

在官网有介绍多种安装方法, 最简单的就是使用pip了:

$ sudo pip install mitmproxy

其他安装方法我没尝试, 详见官网

运行&使用

mitmdump是纯命令行工具, 所以需要通过参数来控制, 简单的使用方法使用参见 说明文档, 由于我自己还没用过, 所以暂时 无法详述

mitmproxy是带交互的, 算是实时监测吧. 运行的方法就是直接

$ mitmproxy

就可以了. 如果要看具体的参数, 可以加上--help查看

然后设置浏览器的代理为127.0.0.1:8080, 这样所有的HTTP请求就会通过mitmproxy过滤后 再发送出去, 如果要修改HTTP的内容, 就要自己写脚本了, 这方面还在研究, 先挖个坑
