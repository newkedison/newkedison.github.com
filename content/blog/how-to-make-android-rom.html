参考资料: ----------------- * * [Initializing a Build Environment](http://source.android.com/source/initializing.html) * [Downloading the Source Tree](http://source.android.com/source/downloading.html) * [Building the System](http://source.android.com/source/building.html) * [[Tutorial] Install Sun Java JDK 6 on Ubuntu 12.04 LTS](http://forum.xda-developers.com/showthread.php?t=2021510) ## 第一步: 安装系统 ---------------------- android的源码推荐是在Ubuntu10.04 64bit上编译, 现在13.04都快要出来了, 还用10.04有点说不过去, 我是用12.04编译的, 也没碰上什么问题. 据官网说, 从Android2.3开始的版本, 都必须在64bit上编译, 所以尽量还是选择64bit的. 下载完系统刻盘或者做U盘启动, 然后安装. 也可以弄个虚拟机来跑, 但是速度估计就很难说了, 注意虚拟机需要有16G的RAM或者Swap, 还要有30G空间. ## 第二步: 安装必要的软件 ------------------- 首先要安装jdk, 目前最新版本是jdk7, 但是教程里面说用jdk6就行, 除非只需要编译2.3以前的版本, 否则jdk5是不行的. 我最后选择了安装jdk6. 按照官网教程里面的方法是安装不了jdk6的, 后来搜到了[参考资料中的那个帖子](http://forum.xda-developers.com/showthread.php?t=2021510), 运行以下命令: $ sudo add-apt-repository "deb http://archive.ubuntu.com/ubuntu hardy main multiverse" $ sudo add-apt-repository "deb http://archive.ubuntu.com/ubuntu hardy-updates main multiverse" $ sudo add-apt-repository "deb http://archive.canonical.com/ lucid partner" $ sudo add-apt-repository "deb http://ppa.launchpad.net/webupd8team/java/ubuntu precise main" $ sudo apt-get update $ sudo apt-get install sun-java6-jdk 这样jdk6就安装好了 下面是安装git等必要的工具, 按照官网说的, 执行以下命令: $ sudo apt-get install git-core gnupg flex bison gperf build-essential \ zip curl libc6-dev libncurses5-dev:i386 x11proto-core-dev \ libx11-dev:i386 libreadline6-dev:i386 libgl1-mesa-glx:i386 \ libgl1-mesa-dev g++-multilib mingw32 openjdk-6-jdk tofrodos \ python-markdown libxml2-utils xsltproc zlib1g-dev:i386 $ sudo ln -s /usr/lib/i386-linux-gnu/mesa/libGL.so.1 /usr/lib/i386-linux-gnu/libGL.so 安装好之后, 官网教程里面还说需要弄什么USB权限, 还要配置ccache, 我暂时都没做, 不影响后面下载和编译源码 ## 第三步: 下载源码 ----------------------- Android的源码是用git托管的, 但是比较复杂, 所以有一个专门的脚本来简化工作, 所以首先是下载这个脚本, 叫repo: $ mkdir ~/bin $ PATH=~/bin:$PATH $ curl https://dl-ssl.google.com/dl/googlesource/git-repo/repo > ~/bin/repo $ chmod a+x ~/bin/repo 然后新建一个文件夹, 用来放Android的源码 $ mkdir WORKING_DIRECTORY $ cd WORKING_DIRECTORY 然后开始下载源码: $ repo init -u https://android.googlesource.com/platform/manifest $ repo sync 第一个命令运行后, 会提示输入名字和邮箱等信息, 可以随便输, 但是如果以后有打算报bug的, 需要填自己的gmail. 第二个命令执行的时间就比较长了, 我花了大概一个下午的时间, 才下载完成, 中途由于网络问题, 导致部分文件没下好, 后来再执行一次`repo sync`就可以了 根据一些地方说的, 也可以自己找别人打包好的源码, 放在上面新建的文件夹里面, 然后用repo sync更新一下, 可以节省时间 ## 第四步: 编译源码 --------------------- 根据官网说的, 执行以下命令 $ . build/envsetup.sh 可以配置好环境, 然后配置编译目标 $ lunch full-eng 最后就是 $ make -j4 四线程开始编译. 整个系统编译下来还算有惊无险, 大概花了1个小时, 顺利完成. {% endraw %} {# vim: ft=markdown #}
